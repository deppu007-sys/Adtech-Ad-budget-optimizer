# Ad Budget Optimizer (Python)

## 📌 Project Overview
This project distributes advertising budget across ads based on their performance (CTR).

## 🚀 Features
- Calculates CTR (Click Through Rate)
- Allocates budget smartly
- Beginner-friendly Python project
- Works on Android (Pydroid 3)

## 🧠 Formula
Budget_i = (CTR_i / Total CTR) * Total Budget

## ▶️ How to Run
1. Open in Pydroid 3
2. Run budget_optimizer.py
3. Enter ad data

## 📊 Example Output
Ad_A → ₹303  
Ad_B → ₹363
